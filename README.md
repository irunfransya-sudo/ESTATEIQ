# EstateIQ

EstateIQ — інформаційно-аналітична система дослідження ринку нерухомості, яка була розроблена для аналізу об’єктів нерухомості, перегляду інформації про житло та взаємодії користувача із системою через сучасний web-інтерфейс.

У проєкті реалізовано backend частину на FastAPI та frontend частину на Vue.js. Система підтримує перегляд каталогу нерухомості, сторінки з детальною інформацією про об’єкти, аналітичну панель, а також взаємодію frontend і backend через API.

Під час розроблення використовувалися Python, FastAPI, SQLite, Vue.js, Vite, JavaScript та CSS. База даних реалізована на SQLite.

Для запуску backend необхідно перейти до папки backend, встановити необхідні залежності та запустити сервер через uvicorn main:app --reload. Frontend запускається окремо через npm run dev у папці frontend.

Проєкт створений у межах кваліфікаційної роботи як приклад web-орієнтованої інформаційної системи дослідження ринку нерухомості.
